# 恐怖故事生成器（疗愈版）- 可运行原型

这是一个**无需安装依赖**的离线网页原型（单文件前端 + LocalStorage），实现了 PRD 的主流程：

- 首页（强度选择 + 免责声明）
- 情绪日记输入（高风险词触发：强制温和 + 提示求助）
- 3 题恐怖意象测试
- 故事配置确认
- 生成中加载页（进度条）
- 故事阅读页（可复制）
- 金钥匙页（核心信念 + 金句 + 3 条可执行建议 + 肯定语 + 情绪温度计）
- 档案页（本地保存/打开记录/清空）

## 运行方式

### 方式 1：直接打开

双击打开 `index.html`（Chrome/Edge/Safari 均可）。

> 注：如果浏览器限制 clipboard API，复制按钮可能需要在 http(s) 环境下运行。

### 方式 2：本地起一个静态服务器（推荐）

```bash
cd /root/.openclaw/workspace/horror-story-generator-static
python3 -m http.server 5173
```

然后打开：

- http://127.0.0.1:5173/
